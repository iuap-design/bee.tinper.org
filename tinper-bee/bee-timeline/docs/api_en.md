## Timeline
常用于当一系列的信息按时间排序的时候或者做个可视化的连接


## API
### Timeline
|Property|Description|Type|Default|
|:---|:-----|:----|:------|
|pending|设置最后一个虚线节点|(boolean|string|ReactNode)|-|
### Timeline.item
|Property|Description|Type|Default|
|:---|:-----|:----|:------|
|color|设置圆环的颜色|(danger|info|news|warning|sucess)|或者其他自定义颜色|-|
|dot|自定义的时间节点|string|ReactNode|或者其他自定义颜色|-|
