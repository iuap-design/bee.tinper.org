# 表单容器 FormGroup

`FormGroup` 组件用来包裹像 `form control` `lable` `help text` `validate state`的元素

## 何时使用

包裹单个表单元素时

## 如何使用

```
import { FormGroup } from 'tinper-bee';

or

import FormGroup from 'bee-form-group';
import bee-form-group/build/FormGroup.css;

```
