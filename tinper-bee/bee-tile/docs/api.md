|参数|说明|类型|默认值|

|Component|最外层包裹元素|html node|'div'|
|border|是否带边框|bool|true|
|children|自定义子内容|object/string/node|-|
|hoverColors|设置hover的样式 oneOf:primary,info,warning,danger,success|strng|-|
